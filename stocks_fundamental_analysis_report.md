## How to read this report

Each holding is graded **A–F** against your fundamental-analysis framework (normalized to /100):

| Grade | Band | Meaning |
|---|---|---|
| **A** | 85–100 | Exceptional — clears business-quality + returns + balance-sheet hurdles cleanly |
| **B** | 70–84 | Good — strong franchise, one or two watch items |
| **C** | 55–69 | Average — mixed signals; quality or returns gap |
| **D** | 40–54 | Below average — weak returns / leverage / earnings stress |
| **F** | <40 | Poor |

**Your stated bias — strong & *rising* FII/DII presence — is weighted heavily** in the shareholding component and called out explicitly in the last column (↑ rising, ↓ falling, → flat).
